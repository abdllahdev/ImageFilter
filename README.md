# ImageFilter
<h2>Image Filter App using Java and JavaFX</h2>
<h4>Features</h4>
<ul>
  <li>Apply multiple filters on the image</li>
  <li>Retreve all filters applied</li>
  <li>Retreve the last filter applied</li>
</ul>

<h4>Filters in this App</h4>
<ul>
  <li>Edge Detector</li>
  <li>Blur</li>
  <li>Emposs</li>
  <li>Sharpen</li>
  <li>Greyscale</li>
</ul>
